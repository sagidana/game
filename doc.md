KEY     MODE    DESCRIPTION                                                         MANA

h       NORMAL  left                                                                -1
j       NORMAL  down                                                                -1
k       NORMAL  up                                                                  -1
l       NORMAL  right                                                               -1

b       NORMAL  half dash to the left                                               -5
B       NORMAL  full dash to the left                                               -10
w       NORMAL  half dash to the right                                              -5
W       NORMAL  full dash to the right                                              -10

CTRL-d  NORMAL  half dash to the bottom                                             -5
CTRL-f  NORMAL  full dash to the bottom                                             -10
CTRL-u  NORMAL  half dash to the top                                                -5
CTRL-b  NORMAL  full dash to the top                                                -10

v       NORMAL  enter select mode

i       NORMAL  attack to the right                                                 -10
a       NORMAL  attack to the left                                                  -10
I       NORMAL  move to nearest opponent to the left and attack                     -15
                if no opponent exist only attack without moving
A       NORMAL  move to nearest opponent to the right and attack                    -15
                if no opponent exist only attack without moving
o       NORMAL  move to nearest opponent above you and attack                       -15
                if no opponent exist enter combat mode without moving
O       NORMAL  move to nearest opponent beneath you and attack                     -15
                if no opponent exist enter combat mode without moving

CTRL-y  NORMAL  move screen one line up                                             -0
CTRL-e  NORMAL  move screen one line down                                           -0
zz      NORMAL  center screen                                                       -0
zt      NORMAL  top screen                                                          -0
zb      NORMAL  bottom screen                                                       -0

:       NORMAL  enter command mode                                                  -0

h       SELECT  select closest element to the left                                  -0
j       SELECT  select closest element beneath                                      -0
k       SELECT  select closest element above                                        -0
l       SELECT  select closest element to the right                                 -0
$       SELECT  select farthest element to the right                                -0
0       SELECT  select farthest element to the left                                 -0
f       SELECT  open letters next to each element to choose that element            -0
